# User Rules Template

This directory contains the **template** for User Rules that should be copied to Cursor Settings.

## Location

- **Template (this directory)**: `.cursor/user-rules/USER_RULES.md` (version-controlled in project)
- **Your Rules**: Cursor Settings → Rules → User Rules (plain text field)

## Available Files

- **USER_RULES.md** - Complete User Rules template (copy content to Cursor Settings)
- **README.md** - This file (quick reference)

## Setup Instructions

1. Open **USER_RULES.md** in this directory
2. Copy the entire content
3. Open Cursor Settings → Rules → User Rules
4. Paste the content in the plain text field
5. Save

## Notes

- User Rules apply to **all projects** you work on
- Project Rules are in `.cursor/rules/` and are project-specific
- User Rules are configured in Cursor Settings, not as files
- The content of **USER_RULES.md** should be pasted into Cursor Settings → Rules → User Rules
